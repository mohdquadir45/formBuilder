# Form Builder
## Steps to Run the Application 
### 1. Clone the Repository
```bash
git clone https://github.com/mohdquadir45/formBuilder.git
cd clone_project for front end and for backend cd clone_project\backend

2. Install Dependencies
Frontend
Navigate to the server\clone_project directory and install dependencies.
cd server\clone_project
npm install

Run the application:
npm run dev

Backend
Navigate to the server\clone_project\backend directory and install dependencies.

cd server\clone_project\backend
npm install
Run the server:
npm run server
